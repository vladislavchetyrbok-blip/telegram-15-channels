# Zodiac Remaining 44 Visuals Import

This guide outlines the generation and import of the 44 remaining visual assets for the 11 Zodiac signs (excluding Aries and the general channel).

## 1. Global Import & Validation Workflow
We highly recommend doing this **one sign at a time**:
1. Generate the 4 images for a single sign using the exact prompts below.
2. Save the 4 images into their exact expected paths within `public/assets/zodiac/`.
3. Run the validator:
   ```bash
   npm run zodiac:assets:validate
   ```
4. Confirm the sign's completeness reads `4/4 | COMPLETE`.
5. Move on to the next sign.

## 2. Global Style & Quality Checklist
- **Negative Prompt (Use for all):** `cartoon, anime, text, typography, letters, watermark, signature, ugly, messy, flat, vector, illustration, cheap, low resolution, crowded, chaotic, neon bright colors, ai artifacts`
- **Rule:** Images must NOT contain visible typography or unreadable symbols. Keep the focus entirely visual, allowing Telegram to add text over the images later if necessary.
